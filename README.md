# Stonk Alert

A website for tracking news, trends, providing information, and more on "meme stocks" and their companies.

##### Requirements

https://nodejs.org/en/ (Version 14.17.1 LTS)
```
$ pip install flask
```


##### Deploy
```
$ cd stonk-alert
$ npm install
$ npm run build
$ cd ..
$ cd flask-server
$ python main.py
```
